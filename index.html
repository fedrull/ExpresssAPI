API Documentation

GET/api/resource

Retrieve information about a resource.

GET /api/resource

Response:

{"id": 1, "Username":" Password"}

GET /api/resource/{id}

Retrieve information about a resource by ID.

GET /api/resource/{id}

Response:

{"id": 1, "Username":" Password"}

POST /api/resource

Create a new resource.

POST /api/resource

Request:

{"Username": "string", "Password": "string"}

Response:

{"id": 2, "string": "string"}

DELETE /api/resource/{id}

Delete a resource by ID.

DELETE /api/resource/{id}

Response:

{"message": "User deleted successfully"}
